Introduction

This module is a payment gateway for PayPal. You can use it for PayPal Standard and PayPal IPN (Instant Payment Notification. With IPN, once the payment is approved by PayPal, all digital products will be sent to buyers automatically. But, be sure to always recheck the payment!

Installing

To install this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Click on Scan New Module.
  3. On 'Available Modules', click 'Documentation' to read this manual.
  4. Click on 'Install' to begin install process.

You may also need to change your template, see Usage section.

[ back to top ]

Usage

This module is intended to be used with Cart Engine or other scripts that support C97's Payment Gateway Module.

You can enable PayPal IPN in module configuration. When using PayPal IPN, once confirmed by PayPal, this module will do the following automatically:

  • Set payment status to Paid.
  • All digital products will be delivered to buyer.
  • An email will be sent to buyer to download the digital products.
  • Order containing digital products only will be set to Delivered.
  • If you do NOT want this to happen automatically, please disable IPN support.

Uninstalling

To uninstall this Module in, follow these easy steps:

  1. Go to Admin CP > Configure Site > Modules.
  2. Find PayPal. Click Uninstall.
  3. You can choose to also remove database & files.

Uninstalling may require you to manually remove some files for 100% clean uninstall.
